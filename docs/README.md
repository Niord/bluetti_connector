# Documentation

This directory contains the public documentation entry points for the repository.

- [Upstream provenance](upstream.md) explains how this work relates to the official BLUETTI Home Assistant integration.
- [Python backend and package](python.md) covers the standalone Python runtime, configuration, and automated checks.
- [Local web page](local-web.md) covers the backend-served browser page and fake-gateway verification flow.
- [Swift package](swift.md) covers `BluettiKit` integration, build, test, OAuth, live updates, and supported commands.
- [macOS sample app](macos-sample.md) covers the SwiftUI menu bar reference app built on top of `BluettiKit`.

The root README is intentionally short. Use these documents when you need module-level setup or verification details.
